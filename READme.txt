Чтобы клиент автоматически запускался при входе в систему, добавьте созданный исполняемый файл в автозагрузку. В Windows это можно сделать следующим образом:
 - Нажмите Win + R, введите shell:startup и нажмите Enter. Это откроет папку автозагрузки.
 - Скопируйте исполняемый файл из папки dist (где находится ваш исполняемый файл) в открытую папку автозагрузки.
Чтобы запустить данный программный продукт сначала нужно:
 - Запустить файл Service_3.py
 - Затем запустить Client_2.py.
   *Также присутсутствует веб интерфейс в файле index_2.html
